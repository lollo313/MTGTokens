Token Tracker

Importa il mazzo, individua i token esatti, tienili sotto controllo durante la partita.

1. Importa il mazzo

Incolla la decklist (una carta per riga, il numero iniziale è opzionale). Basename e set code vengono ignorati.

Oppure importa da un ID mazzo Archidekt (sperimentale)

L'API di Archidekt non è ufficialmente documentata e potrebbe bloccare le richieste dirette dal browser (CORS). Se non funziona, usa il metodo testuale sopra: copia la decklist da Archidekt e incollala qui.

2. Token della partita

Nessun mazzo caricato. Incolla una decklist sopra per iniziare.
